Tabletop RPG Character Creator

Welcome to Tabletop RPG Character Creator! With this app you can easily create and store basic character data to use in tabletop games. Once you fill out the short form, select create character to add it to the database. Stored characters can be editted and deleted.

Note: This early build is not a final product and is intended to demonstrate basic usage of Node, Express, and Mongoose to build a front to back end app with basic CRUD functionality. Future builds will integrate this app with a suite of other apps for a custom homebrew game. Follow my Github repo here to check out planned and in-progress features.
